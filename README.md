Apex Games Store
Uma loja virtual focada na venda de keys digitais para jogos, oferecendo uma experiência moderna para explorar lançamentos, tendências e promoções em diferentes plataformas.

Sobre o projeto
A Apex Games Store foi desenvolvida com o objetivo de simular uma plataforma de e-commerce gamer, permitindo aos usuários visualizar jogos, promoções e futuras funcionalidades para compra de keys digitais.
O projeto possui interface inspirada em lojas modernas de games, com foco em: Navegação intuitiva, organização dos jogos por categorias, layout responsivo, destaque para lançamentos e tendências, experiência visual voltada ao público gamer
